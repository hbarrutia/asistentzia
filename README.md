
# Generador de Asistentzia

Esta aplicación permite subir una plantilla y un fichero de origen, introducir horas por módulo y descargar el archivo final.

## Cómo desplegar en Streamlit Cloud

1. Sube estos archivos a un repositorio de GitHub:
   - `asistentzia_app.py`
   - `requirements.txt`

2. Ve a [https://streamlit.io/cloud](https://streamlit.io/cloud) y conecta tu cuenta de GitHub.

3. Selecciona el repositorio y el archivo `asistentzia_app.py` como archivo principal.

4. ¡Listo! Tu app estará disponible online.
